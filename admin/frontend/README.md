## 🚀 环境准备

请确保本地已安装以下环境：
- **nvm**：Node 版本管理工具（推荐使用）
- **Node.js**：`v18.19.1`
- **Yarn**：作为包管理器（推荐使用）

---

## 🧩 前端运行步骤

```bash
# 进入项目目录

# 使用指定 Node 版本
nvm use 18.19.1

# 安装依赖（建议使用国内镜像） 如果未安装 yarn，可执行：npm install -g yarn
yarn --registry=https://registry.npmmirror.com

# 启动开发环境
yarn dev
```
